# Sistema de Programação Aplicado em Hardware # 
 

 ***Descrição:***

O objetivo é o desenvolvimento de um sistema de programação aplicado em Hardware, como requisitos básicos estão a utilização da placa Arduino e a programação
em linguagem C.

Neste caso trata-se de um sistema simples para monitorar a umidade do solo, caso este se torne muito seco, o sistema acende uma luz de alerta.
